# Terraform

## Step 0: Initialize Terraform

terraform init

## Step 1: Plan Resources

terraform plan -var-file="variables-us-east-2.tfvars"

## Step 2: Apply Resources

terraform apply -var-file="variables-us-east-2.tfvars"

## Step 3: Destroy Resources

terraform apply -var-file="variables-us-east-2.tfvars"
